# Gift Ideas App

Never forget the perfect gift. A web app that tracks people, reminds you of upcoming birthdays, and generates personalised gift ideas using AI.

## Features

- People database with names, relationships, birthdays and interests
- Daily email notifications for birthdays in the next 30 days
- AI-powered personalised gift suggestions via Claude API
- Secure: each user only sees their own data

## Tech Stack

- React
- Supabase (database, auth, edge functions)
- Anthropic Claude API
- Vercel (hosting + serverless functions)
- Resend (email notifications)
